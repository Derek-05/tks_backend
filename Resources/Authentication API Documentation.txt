User Authentication & Profile Management API Documentation
Base URL: /api/auth
Endpoints:

Register a New User
* URL: /signup
* Method: POST
* Description: Registers a new user.
* Request Body:
json
  {
    "email": "user@example.com",
    "password": "password123"
}
* Success Response:
* Code: 201 Created
* Content:
json
 
{
    "success": true,
    "user": {
        "id": 1,
        "email": "user@example.com"
        // Other user attributes
    }

* Error Response:
* Code: 400 Bad Request
* Content:
json

{
    "success": false,
    "error": "Email already registered"
}



Login

* URL: /signin
* Method: POST
* Description: Logs in a user.
* Request Body:
json
  {
    "email": "user@example.com",
    "password": "password123"
}
* Success Response:
* Code: 200 OK
* Content:
json
 
{
    "success": true,
    "token": "JWT_TOKEN",
    "user": {
        "id": 1,
        "email": "user@example.com"
        // Other user attributes (excluding password)
    }

* Error Response:
* Code: 400 Bad Request
* Content:

json
{
    "success": false,
    "error": "Invalid credentials"
}




Logout
* URL: /logout
* Method: GET
* Description: Logs out a user by clearing the authentication token.
* Success Response:
* Code: 200 OK
* Content:

json

{
    "success": true,
    "message": "Logged out"
}


Get User Profile
* URL: /profile
* Method: GET
* Description: Retrieves the profile of the authenticated user.
* Authentication: Required (JWT token)
* Success Response:
* Code: 200 OK
* Content:
json
 
{
    "success": true,
    "user": {
        "id": 1,
        "email": "user@example.com"
        // Other user attributes (excluding password)
    }

* Error Response:
* Code: 404 Not Found
* Content:
json
{
    "success": false,
    "error": "User not found"
}
