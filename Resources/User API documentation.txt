Base URL: localhost:8080/api
Endpoints
Get All Users

* URL: /allusers
* Method: GET
* Description: Retrieves a paginated list of users.
* Query Parameters:
* pageNumber (optional): Specifies the page number (default: 1)
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "data": {
        "users": [array_of_users],
        "page": current_page_number,
        "pages": total_number_of_pages
    }
}



Get Single User
* URL: /singleuser/:id
* Method: GET
* Description: Retrieves a single user by their ID.
* Path Parameters:
* id: ID of the user to retrieve
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "data": {
        "user": {user_object}
    }
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "User not found"
}



Update User
* URL: /edituser/:id
* Method: PUT
* Description: Updates an existing user.
* Path Parameters:
* id: ID of the user to update
* Request Body:
* Data to update the user with
* Response:
* Status Code: 200 OK
* Body:
json
{
    "success": true,
    "data": {
        "updatedUser": {updated_user_object}
    }
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "User not found"
}



Delete User
* URL: /admin/user/delete/:id
* Method: DELETE
* Description: Deletes an existing user.
* Path Parameters:
* id: ID of the user to delete
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "message": "User deleted"
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "User not found"
}

* Error Responses
* Status Code: 400 Bad Request
* Body:
json{
    "success": false,
    "error": "Invalid page number"
}
* Status Code: 500 Internal Server Error
* Body:
json{
    "success": false,
    "error": "Internal Server Error"}
