Job Offering Management API
Base URL: localhost:8080/api
Endpoints
Get All Job Offerings
* URL: /allJobs
* Method: GET
* Description: Retrieves a paginated list of job offerings.
* Query Parameters:
* keyword (optional): Search keyword to filter job offerings by title.
* pageNumber (optional): Specifies the page number (default: 1)
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "jobOfferings": [array_of_job_offerings],
    "page": current_page_number,
    "pages": total_number_of_pages
}



Get Single Job Offering
* URL: /job/:id
* Method: GET
* Description: Retrieves a single job offering by its ID.
* Path Parameters:
* id: ID of the job offering to retrieve
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "jobOffering": {job_offering_object}
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "Job not found"
}



Create Job Offering
* URL: /createJob
* Method: POST
* Description: Creates a new job offering.
* Request Body:
* title: Title of the job offering
* description: Description of the job offering
* salary: Salary for the job offering
* qualifications: Qualifications required for the job offering
* available: Availability status of the job offering
* applicationDeadline: Application deadline for the job offering
* Response:
* Status Code: 201 Created
* Body:
json{
    "success": true,
    "newJobOffering": {created_job_offering_object}
}



Update Job Offering
* URL: /updateJob/:id
* Method: PUT
* Description: Updates an existing job offering by its ID.
* Path Parameters:
* id: ID of the job offering to update
* Request Body:
* Data to update the job offering with
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "data": {
        "updatedJob": {updated_job_offering_object}
    }
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "Job not found"
}



Delete Job Offering
* URL: /deleteJob/:id
* Method: DELETE
* Description: Deletes an existing job offering by its ID.
* Path Parameters:
* id: ID of the job offering to delete
* Response:
* Status Code: 200 OK
* Body:
json{
    "success": true,
    "message": "Job deleted"
}
* Error Response:
* Status Code: 404 Not Found
* Body:
json{
    "success": false,
    "error": "Job not found"
}

* Error Responses
* Status Code: 400 Bad Request
* Body:
json

{
    "success": false,
    "error": "Invalid page number"
}
?  Status Code: 500 Internal Server Error
Body:
json{
    "success": false,
    "error": "Internal Server Error"
}
